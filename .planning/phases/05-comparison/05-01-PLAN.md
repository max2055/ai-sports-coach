<task type="auto">
  <name>Task 2: Create Pro Video API</name>
  <files>api/models/pro.py, api/services/pro_service.py, api/routers/pros.py, api/main.py</files>
  <action>
    1. 创建 `api/models/pro.py`:
       ```python
       from pydantic import BaseModel
       from typing import Literal

       ProVideoType = Literal["forehand", "backhand", "serve", "volley"]

       class ProVideo(BaseModel):
           id: str
           type: ProVideoType
           name: str
           description: str
           file: str
           duration: float | None = None

       class ProVideoList(BaseModel):
           videos: list[ProVideo]
       ```

    2. 创建 `api/services/pro_service.py`:
       - PROS_DIR = Path("static/pros")
       - list_pro_videos(video_type=None) → list[ProVideo]
         * 读取 manifest.json
         * 可按类型筛选
         * 使用 ffprobe 提取时长
       - get_pro_video(video_id) → ProVideo
         * 返回单个视频信息
       - get_pro_video_path(video_id) → Path
         * 返回视频文件路径

    3. 创建 `api/routers/pros.py`:
       ```python
       router = APIRouter()

       @router.get("/pros", response_model=ProVideoList)
       async def list_pros(type: ProVideoType = None):
           return {"videos": list_pro_videos(type)}

       @router.get("/pros/{video_id}", response_model=ProVideo)
       async def get_pro(video_id: str):
           return get_pro_video(video_id)

       @router.get("/pros/{video_id}/file")
       async def get_pro_file(video_id: str):
           path = get_pro_video_path(video_id)
           return FileResponse(path)
       ```

    4. 更新 `api/main.py`:
       - 导入 pros router
       - app.include_router(pros.router, prefix="/api", tags=["pros"])
       - 挂载 static 目录: app.mount("/static", StaticFiles(directory="static"), name="static")
  </action>
