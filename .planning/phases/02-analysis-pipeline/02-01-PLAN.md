    1. 创建 `api/routers/analysis.py`:
       ```python
       from fastapi import APIRouter, BackgroundTasks, HTTPException
       from models.analysis import AnalysisState, AnalysisResponse
       from services.analysis_service import run_analysis, get_analysis_status

       router = APIRouter()

       @router.post("/analyze/{video_id}", response_model=AnalysisResponse)
       async def start_analysis(video_id: str, background_tasks: BackgroundTasks):
           # 验证视频文件存在
           # 启动后台分析任务
           background_tasks.add_task(run_analysis, video_id, video_path, analysis_type)
           return {"video_id": video_id, "status": "pending", "progress": 0, "message": "Analysis started"}

       @router.get("/status/{video_id}", response_model=AnalysisState)
       async def get_status(video_id: str):
           return get_analysis_status(video_id)
       ```

    2. 更新 `api/main.py`:
       - 导入 analysis router
       - app.include_router(analysis.router, prefix="/api", tags=["analysis"])
