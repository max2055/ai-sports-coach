  <name>Task 1: Install ECharts and Create API Client</name>
  <files>web/package.json, web/src/api/charts.ts</files>
  <action>
    1. 安装 ECharts:
       ```bash
       cd web && npm install echarts
       ```

    2. 创建 `web/src/api/charts.ts`:
       ```typescript
       import axios from 'axios'
       import type { AnalysisType } from '../types/upload'

       const API_BASE = import.meta.env.VITE_API_BASE_URL || 'http://localhost:8000'

       export interface ServeHeightPoint {
         frame_number: number
         time_seconds: number
         height_pct: number
       }

       export interface HitPoint {
         frame_number: number
         x_pct: number
         y_pct: number
         issue_type?: string
       }

       export interface RadarDimension {
         name: string
         value: number
       }

       export interface IssueStat {
         issue_type: string
         count: number
         severity: 'high' | 'medium' | 'low'
       }

       export async function getServeHeightData(videoId: string): Promise<{ points: ServeHeightPoint[] }> {
         const res = await axios.get(`${API_BASE}/api/charts/${videoId}/serve-height`)
         return res.data
       }

       export async function getHitPointData(videoId: string): Promise<{ points: HitPoint[] }> {
         const res = await axios.get(`${API_BASE}/api/charts/${videoId}/hit-points`)
         return res.data
       }

       export async function getRadarData(videoId: string): Promise<{ dimensions: RadarDimension[] }> {
         const res = await axios.get(`${API_BASE}/api/charts/${videoId}/radar`)
         return res.data
       }

       export async function getIssueStats(videoId: string): Promise<{ stats: IssueStat[] }> {
         const res = await axios.get(`${API_BASE}/api/charts/${videoId}/issues`)
         return res.data
       }
       ```

    创建四个图表组件，每个使用 ECharts:

    1. `ServeHeightChart.vue`:
       - Props: points (ServeHeightPoint[])
       - ECharts line chart
       - xAxis: time_seconds
       - yAxis: height_pct (反转，越低表示越高)
       - 标题: "发球高度趋势"

    2. `HitPointHeatmap.vue`:
       - Props: points (HitPoint[])
       - ECharts scatter/heatmap
       - 使用网球场背景图（可选）
       - 点颜色: 有问题=红色，正常=蓝色
       - 标题: "击球点分布"

    3. `ConsistencyRadar.vue`:
       - Props: dimensions (RadarDimension[])
       - ECharts radar chart
       - 6 维度: 击球技术、步法、旋转、节奏、体能、战术
       - 最大值: 10
       - 标题: "动作一致性评分"

    4. `IssueStatsChart.vue`:
       - Props: stats (IssueStat[])
       - ECharts bar/pie chart
       - 柱状图显示问题类型出现次数
       - 颜色按严重程度: high=红, medium=橙, low=黄
       - 标题: "问题类型统计"

    每个组件:
       - 挂载时初始化 ECharts 实例
       - watch props 变化更新图表
       - resize 时重新设置尺寸
